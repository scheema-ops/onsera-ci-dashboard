## What's Included

- **20 competitors** across 4 market segments
- **Weekly digest** with latest market intelligence
- **Evidence grading** (GREEN/YELLOW/RED) for competitor claims
- **They Say / We Say** sales positioning guides
- **Win/Lose scenarios** for each competitor

## Market Segments

| Segment | Description |
|---------|-------------|
| PBM-Aligned | Embedded in pharmacy benefit; formulary control |
| Coaching-First | Behavior change focus; scalable engagement |
| Clinical-First | Physician-led; medication management |
| Hybrid | Prescribing + coaching; outcomes focus |

## Updating the Dashboard

1. Open the live dashboard
2. Press `Ctrl+Shift+A` to open Admin Mode
3. Edit the JSON data
4. Click "Export HTML"
5. Rename to `index.html`
6. Upload to this repository (replaces existing file)
7. Netlify auto-deploys in ~1 minute

## Admin Mode Access

- **Keyboard:** `Ctrl+Shift+A`
- **Mouse:** Click version number 3 times quickly
